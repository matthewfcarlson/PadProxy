# Case / Enclosure

3D printable enclosure designs for PadProxy.

## Variants

- **Internal mount** - Mounts inside PC case (primary design)
  - Designed for standard PC case mounting holes
  - Cable management features
  - Antenna routing channel

- **External** - Standalone desktop enclosure (optional)

## Files

- STL files for 3D printing
- STEP files for modification
- Source files (FreeCAD/OpenSCAD)

## Printing Guidelines

- Material: PLA or PETG recommended
- Layer height: 0.2mm
- Infill: 20%
- Supports: May be required for some features

## Mounting

The internal mount variant is designed to:
- Attach to standard PC case mounting points
- Route antenna cable to rear I/O area
- Allow easy access to status LEDs
